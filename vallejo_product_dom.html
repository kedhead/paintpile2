True Metallic Metal Sets

The 4-color sets offer a compact and organized way to work with each chromatic family.

Each set includes the four tones that compose it – BASE, LIGHT, SHADE and AIRBRUSH – providing a complete and logical solution for painting metals in a specific hue.

Each set is designed within the BSL system structure, ensuring consistent, reproducible results adaptable to any painting style.
